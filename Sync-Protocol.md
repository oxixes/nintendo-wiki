| ID | Description |
| --- | --- |

| 0x02 | End message |
| 0x03 | Start end message |

| 0x11 | Start ack |
| 0x12 | End ack |
| 0x13 | Start end ack |
| 0x14 | End alone ack |
| 0x21 | Data message (raw) |
| 0x22 | Data message (compressed) |
